# Amazon Clone

Fully functional Amazon Clone with responsive design, product catalog, Add to Cart, Return & Orders, delivery selection, and payment options including Cash on Delivery & online payments. Realistic UI mimics Amazon, with interactive tabs, dynamic product listing, and order management for a seamless shopping experience.

## Features

- Responsive design
- Add to Cart functionality
- Return & Orders tab
- Delivery selection
- Payment options (COD & Online)
- Interactive product catalog
